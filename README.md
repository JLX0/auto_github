# Auto-Github
Automatically complete tasks related to Github with LLMs etc.

# Requirements

OS: Linux-based

Python: >= 3.10

# Installation

`git clone https://github.com/JLX0/auto_github.git`

`cd auto_github`

1. `pip install -e .` (for installation in editable mode)

2. `pip install .` (for ordinary installation)
